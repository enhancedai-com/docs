The **AI Text Generator** node is the core component for generating content via Artificial Intelligence. It allows interfacing with multiple LLM providers, offering granular control over AI behavior, web search capabilities, and memory persistence.

## Input fields

These fields define the structure of the request sent to the model:

- **Prompt:** The main command or question sent to the AI.
- **Context:** Supporting information or knowledge base to ground the response.
- **Instructions:** Specific rules regarding behavior, formatting, and constraints.
- **Persona:** Defines the tone of voice and professional profile the AI should adopt.

## Provider settings

### OpenAI

- **Auto Settings:** If `True`, sets `outputTokenMax` to 0. If `False`, allows manual configuration.
- **Temperature:** 0 to 2 (Creativity).
- **F-Penalty (Frequency):** 0 to 2.
- **P-Penalty (Presence):** 0 to 2.

### Groq

- **Auto Settings:** If `True`, `max tokens` = 0. If `False`, allows manual configuration.
- **Seed:** -1 to 100.
- **Temperature:** 0 to 1.
- **Top P:** 0 to 1.
- **Groq Stop:** Field to input custom stop sequence strings.

### Anthropic & Gemini

- **Auto Settings:** If `True`, sets `output token max` to 0.
- **Top K:** 1 - 40.
- **Top P:** 1 - 40 (Anthropic) / 0 - 1 (Gemini).
- **Temperature:** 0 - 1.

### Perplexity

- **Search Context Size:** Low, Medium, High.
- **Auto Settings:** If `True`, sets `output token max` to 0.
- **Top K / Top P:** 1 - 40 / 0 - 1.
- **Temperature:** 0 - 2.

### X AI

- **AI Temperature:** 0 - 2.
- **Auto Settings:** If `True`, sets `output token max` to 0.

## Model compatibility

**Web Search** and **Multi-Modal** options are contextual and only appear in the UI if the selected model supports them.

## Advanced features

### Streaming setting

Enables real-time response generation. When active, you must select the output type:

- **Chat**
- **Form**
- **API Call-back**

### Execution management

- **Clear Output:** Every time a new execution starts, the output, links, and logs are automatically cleared.
- **Link and File Type:** If enabled, you must select a file type to transform the AI output into a document. A direct download link is then provided to the user.

### Node outputs

1. **Output:** The generated text content.
2. **Logs:** Technical execution data and audit trails.
3. **Link:** URL for the generated file (if file conversion is active).

## Session memory

Managed based on the session defined in the **Global Node**, memory stores inputs and outputs to be passed as future context.

### Context passing options

- Pass all stored information.
- Pass the **First X** or **Last X** entries.
- Pass the **First X** or **Last X** words of the information.

### Maintenance

A dedicated option is available to clear all stored session history.
