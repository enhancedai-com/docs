# Documentation

Project documentation using [Mintlify](https://mintlify.com).

## About Mintlify

Mintlify is a modern platform for creating and hosting technical documentation. It offers interactive components, API reference support, and an optimized writing experience.

## Requirements

The Mintlify CLI requires **Node.js v20.17.0 or higher**. It's recommended to use an LTS version for better stability.

## Installation

Install the CLI globally using npm or pnpm:

```bash
npm i -g mint
```

or

```bash
pnpm add -g mint
```

## Running locally

Run the following command from the documentation directory root (where the `docs.json` file is located):

```bash
mint dev
```

The documentation will be available at: **http://localhost:3000**
