## O que isso habilita

Dentro do n8n você pode construir workflows que automaticamente:

- Buscam a lista dos seus agentes FluxPrompt
- Inspecionam a estrutura de nodes de um agente
- Ramificam a lógica com base nos agentes existentes

## Pré-requisitos

- Instância n8n (cloud ou self-hosted, v1.22+)
- Um **service token** de workspace do FluxPrompt (veja [Autenticação](/mcp/pt-br/authentication)) — gere um em [**Settings → MCP Tokens**](https://app.fluxprompt.ai/dashboard-beta?settings=mcpTokens). Um service token estático é o método recomendado para o n8n.

## Passo a passo

<Steps>
  <Step title="Adicione um node MCP Client Tool">
    No seu workflow n8n, clique em **+** para adicionar um node. Pesquise por **MCP Client Tool** e selecione-o.
  </Step>
  <Step title="Configure a conexão">
    Nas configurações do node, defina:

    | Campo | Valor |
    |-------|-------|
    | **Server URL** | `https://mcp.fluxprompt.ai/mcp` |
    | **Authentication** | Header Auth |
    | **Header Name** | `Authorization` |
    | **Header Value** | `Bearer YOUR_SERVICE_TOKEN` |

    Substitua `YOUR_SERVICE_TOKEN` pelo seu service token real (começa com `mcp_`).
  </Step>
  <Step title="Escolha a tool e os parâmetros">
    No dropdown **Tool**, selecione entre:

    - `list_agents` — nenhum parâmetro necessário
    - `get_agent` — requer `agentId` (o UUID do agente)
    - `list_node_types` — nenhum parâmetro necessário

    Defina os parâmetros obrigatórios na seção **Tool Parameters**.
  </Step>
  <Step title="Teste o node">
    Clique em **Execute node**. A saída conterá a resposta estruturada do FluxPrompt.
  </Step>
</Steps>

## Exemplo: listar todos os agentes e filtrar por nome

Este workflow lista todos os agentes FluxPrompt e filtra os que têm "Parser" no nome:

1. **MCP Client Tool** — chama `list_agents`
2. **Code node** — `return $input.first().json.agents.filter(a => a.name.includes('Parser'))`
3. **O que você precisar a seguir** — envie os resultados para o Slack, escreva em uma planilha, etc.
