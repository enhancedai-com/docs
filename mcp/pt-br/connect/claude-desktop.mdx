title: "Conectar Claude Desktop"
description: "Adicione o FluxPrompt ao Claude Desktop para poder perguntar ao Claude sobre os agentes do seu workspace."
---

## O que isso habilita

Uma vez conectado, você pode perguntar ao Claude coisas como:

- *"Quais agentes tenho no FluxPrompt?"*
- *"Mostre a estrutura do meu agente Lead Scorer"*
- *"Quais tipos de node estão disponíveis no FluxPrompt?"*

O Claude chamará as tools do FluxPrompt MCP automaticamente e exibirá os resultados inline.

<Warning>
**O Claude Desktop não aceita tokens colados.** Ele rejeita tokens bearer fornecidos pelo usuário, então o FluxPrompt conecta via **OAuth** — você faz login por uma tela de consentimento do FluxPrompt e nunca manipula um token. Não tente adicionar um header `Authorization` em um arquivo de configuração; não vai funcionar. Veja [Autenticação](/mcp/pt-br/authentication).
</Warning>

## Pré-requisitos

- Claude Desktop instalado ([baixar aqui](https://claude.ai/download))
- Uma conta FluxPrompt com acesso a pelo menos um workspace

## Passo a passo

<Steps>
  <Step title="Abra as configurações de conectores">
    No Claude Desktop, vá em **Settings → Connectors** e clique em **Add custom connector**.
  </Step>
  <Step title="Informe a URL do servidor FluxPrompt">
    Cole a URL do servidor FluxPrompt MCP:

    ```
    https://mcp.fluxprompt.ai/mcp
    ```

    Dê ao conector um nome reconhecível (ex.: `FluxPrompt`) e confirme.
  </Step>
  <Step title="Faça login e conceda acesso">
    O Claude abre uma tela de login + consentimento do FluxPrompt. Faça login com sua conta FluxPrompt, **selecione qual workspace** você quer liberar e aprove.

    Nenhum token está envolvido — a autenticação é feita inteiramente por esse fluxo OAuth.
  </Step>
  <Step title="Verifique a conexão">
    Em uma nova conversa, digite:

    > *Liste meus agentes FluxPrompt*

    O Claude deve responder com a lista de agentes do seu workspace. Se aparecer um erro, consulte a página de [Solução de problemas](/mcp/pt-br/troubleshooting).
  </Step>
</Steps>

## O que o Claude pode fazer com o FluxPrompt

Após a conexão, o Claude tem acesso a estas tools:

| Tool | O que faz |
|------|-----------|
| `list_agents` | Lista todos os agentes do seu workspace |
| `get_agent` | Busca a estrutura completa de um agente específico |
| `list_node_types` | Lista todos os tipos de node disponíveis no FluxPrompt |

O Claude escolherá a tool correta automaticamente com base na sua pergunta. Você também pode pedir explicitamente: *"Use `get_agent` para mostrar a estrutura do agente de ID abc123"*.

## Solução de problemas

Se a conexão não estiver funcionando, consulte a página de [Solução de problemas](/mcp/pt-br/troubleshooting). Os problemas mais comuns são:

- Tentar usar um token colado em vez do fluxo OAuth (o Claude Desktop só suporta OAuth)
- Não concluir a tela de login + consentimento do FluxPrompt
- Selecionar o workspace errado durante o consentimento
