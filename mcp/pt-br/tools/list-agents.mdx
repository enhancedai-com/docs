---

## Descrição

`list_agents` retorna todos os agentes (flows) do seu workspace aos quais o seu token tem acesso. A resposta inclui o ID, nome, descrição, quantidade de nodes e URL pública de cada agente — tudo que você precisa para referenciar ou inspecionar um agente.

## Entrada

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `search` | string | Não | Filtro de substring (case-insensitive) sobre o nome do agente. Omita para retornar todos os agentes. |
| `cursor` | string | Não | Cursor de paginação opaco, retornado como `nextCursor` por uma chamada anterior. Omita para buscar a primeira página. Trate o valor como opaco — não o interprete, não o construa e não o reutilize entre ferramentas. |

Nenhum parâmetro é obrigatório. Chamar `list_agents` sem entrada retorna a primeira página de todos os agentes do seu workspace. O servidor controla o tamanho da página.

## Saída

```json
{
  "agents": [
    {
      "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "name": "Lead Scorer",
      "description": "Scores inbound leads based on company size and tech stack",
      "updatedAt": "2026-05-31T14:22:00Z",
      "nodeCount": 12,
      "isPublic": false
    },
    {
      "id": "b2c3d4e5-f6a7-8901-bcde-f12345678901",
      "name": "Invoice Parser",
      "description": null,
      "updatedAt": "2026-06-01T09:45:00Z",
      "nodeCount": 7,
      "isPublic": true
    }
  ],
  "total": 19,
  "nextCursor": "eyJvIjo1MH0"
}
```

### Campos da saída

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `agents` | array | Lista de agentes desta página. |
| `agents[].id` | string (UUID) | Identificador único. Use este valor com `get_agent`. |
| `agents[].name` | string | Nome de exibição do agente conforme mostrado na interface do FluxPrompt. |
| `agents[].description` | string \| null | Descrição do agente, ou `null` se nenhuma estiver definida. |
| `agents[].updatedAt` | string ISO 8601 | Quando o agente foi modificado pela última vez. |
| `agents[].nodeCount` | number | Número de nodes na versão atual do agente. |
| `agents[].isPublic` | boolean | Se o agente tem um endpoint público. |
| `total` | number | Número total de agentes que correspondem à consulta. Informativo — use `nextCursor` para conduzir a paginação. |
| `nextCursor` | string | Cursor opaco para a próxima página. **Presente apenas quando há mais resultados; omitido na última página.** |

## Paginação

`list_agents` usa paginação por cursor opaco ([MCP 2025-11-25](https://modelcontextprotocol.io)). O servidor decide o tamanho da página. Para percorrer os resultados, chame novamente passando o `nextCursor` anterior como `cursor` e pare quando `nextCursor` estiver ausente:

```
Chamada 1: list_agents({})                        → página 1, nextCursor = "eyJvIjo1MH0"
Chamada 2: list_agents({ cursor: "eyJvIjo1MH0" }) → página 2, nextCursor = "eyJvIjoxMDB9"
Chamada 3: list_agents({ cursor: "eyJvIjoxMDB9" })→ página 3, sem nextCursor → última página
```

<Warning>
Os valores de cursor são opacos. Não os interprete nem os construa, e nunca passe um cursor retornado por uma ferramenta diferente. Um cursor malformado ou inválido retorna o erro JSON-RPC `-32602` (Invalid params) com `data: { "invalid": "cursor" }`.
</Warning>

## Exemplos de prompts

Você pode perguntar ao seu cliente de IA em linguagem natural — ele chamará `list_agents` automaticamente:

- *"Mostre todos os meus agentes FluxPrompt"*
- *"Liste os agentes no FluxPrompt e me diga quais são públicos"*
- *"Quantos agentes tenho no FluxPrompt?"*
