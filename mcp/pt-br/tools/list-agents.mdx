title: "list_agents"
description: "Lista todos os agentes do seu workspace FluxPrompt."
---

## Descrição

`list_agents` retorna todos os agentes (flows) do seu workspace aos quais o seu token tem acesso. A resposta inclui o ID, nome, descrição, quantidade de nodes e URL pública de cada agente — tudo que você precisa para referenciar ou inspecionar um agente.

## Entrada

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `search` | string | Não | Filtro de substring (case-insensitive) sobre o nome do agente. Omita para retornar todos os agentes. |
| `limit` | number | Não | Número máximo de agentes a retornar. Padrão: 20. Máximo: 200. |
| `offset` | number | Não | Número de agentes a pular, para paginação. Padrão: 0. |

Nenhum parâmetro é obrigatório. Chamar `list_agents` sem entrada retorna a primeira página de todos os agentes do seu workspace.

## Saída

```json
{
  "agents": [
    {
      "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "name": "Lead Scorer",
      "description": "Scores inbound leads based on company size and tech stack",
      "updatedAt": "2026-05-31T14:22:00Z",
      "nodeCount": 12,
      "isPublic": false
    },
    {
      "id": "b2c3d4e5-f6a7-8901-bcde-f12345678901",
      "name": "Invoice Parser",
      "description": null,
      "updatedAt": "2026-06-01T09:45:00Z",
      "nodeCount": 7,
      "isPublic": true
    }
  ],
  "total": 19
}
```

### Campos da saída

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `agents` | array | Lista de agentes. |
| `agents[].id` | string (UUID) | Identificador único. Use este valor com `get_agent`. |
| `agents[].name` | string | Nome de exibição do agente conforme mostrado na interface do FluxPrompt. |
| `agents[].description` | string \| null | Descrição do agente, ou `null` se nenhuma estiver definida. |
| `agents[].updatedAt` | string ISO 8601 | Quando o agente foi modificado pela última vez. |
| `agents[].nodeCount` | number | Número de nodes na versão atual do agente. |
| `agents[].isPublic` | boolean | Se o agente tem um endpoint público. |
| `total` | number | Número total de agentes que correspondem à consulta (ignorando `limit`/`offset`), para você saber quantas páginas faltam. |

## Paginação

`list_agents` usa paginação baseada em `offset`. Compare `total` com `limit` + `offset` para saber se ainda há agentes, e então avance o `offset`:

```
Chamada 1: list_agents({ limit: 20 })              → agentes 1-20,  total = 45
Chamada 2: list_agents({ limit: 20, offset: 20 })  → agentes 21-40, total = 45
Chamada 3: list_agents({ limit: 20, offset: 40 })  → agentes 41-45, total = 45
```

## Exemplos de prompts

Você pode perguntar ao seu cliente de IA em linguagem natural — ele chamará `list_agents` automaticamente:

- *"Mostre todos os meus agentes FluxPrompt"*
- *"Liste os agentes no FluxPrompt e me diga quais são públicos"*
- *"Quantos agentes tenho no FluxPrompt?"*
