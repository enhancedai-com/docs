## Description

`list_node_types` returns the catalogue of every node type available in FluxPrompt — what each one does, what inputs it accepts, and what outputs it produces. Use this to understand the building blocks an agent is made of.

## Input

No parameters required.

## Output

```json
{
  "nodeTypes": [
    {
      "type": "commandTextNode",
      "label": "Command Text Node",
      "description": "Executes a text command or prompt and passes it downstream.",
      "category": "AI",
      "inputSchema": {
        "type": "object",
        "properties": {
          "model": { "type": "string", "description": "AI model to use" },
          "command": { "type": "string", "description": "Text command or prompt to execute" }
        },
        "required": ["command"]
      },
      "outputSchema": {
        "type": "object",
        "properties": {
          "text": { "type": "string", "description": "Generated text output" }
        }
      }
    },
    {
      "type": "conditionNode",
      "label": "Condition Node",
      "description": "Evaluates a condition and routes flow execution along the true or false branch.",
      "category": "Logic",
      "inputSchema": { "..." : "..." },
      "outputSchema": { "..." : "..." }
    }
  ]
}
```

### Output fields

| Field | Type | Description |
|-------|------|-------------|
| `nodeTypes` | array | Full list of node types. |
| `nodeTypes[].type` | string | Machine-readable identifier (camelCase, e.g. `commandTextNode`). Matches the `type` field in `get_agent` node objects. |
| `nodeTypes[].label` | string | Human-readable name as shown in the FluxPrompt canvas. |
| `nodeTypes[].description` | string | What this node does. |
| `nodeTypes[].category` | string | Grouping: `Input/Output`, `AI`, `Data`, `Logic`, `Utility`, `Integration`. |
| `nodeTypes[].inputSchema` | JSON Schema | The configuration this node accepts. |
| `nodeTypes[].outputSchema` | JSON Schema | The data this node produces for downstream nodes. |
