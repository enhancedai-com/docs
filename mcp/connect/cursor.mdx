## What this enables

Once connected, the Cursor AI assistant can answer questions about your FluxPrompt agents without leaving your editor:

- *"What agents do I have set up for lead enrichment?"*
- *"Show me the nodes in the Email Parser agent so I can replicate the logic here"*

## Prerequisites

- Cursor installed ([download here](https://cursor.sh))
- A FluxPrompt workspace **service token** (see [Authentication](/mcp/authentication))

<Note>
Cursor can also authenticate via **OAuth** (add the server URL and complete the FluxPrompt login + consent flow), the same as Claude Code. This guide uses a static service token, which is handy for shared or headless setups. Either method works.
</Note>

## Step-by-step

<Steps>
  <Step title="Open Cursor settings">
    Press `Cmd/Ctrl + Shift + P` to open the command palette, type `MCP`, and select **Cursor Settings → MCP**.

    Alternatively, go to **Settings → Features → MCP Servers**.
  </Step>
  <Step title="Add a new MCP server">
    Click **Add new MCP server** and fill in the fields:

    | Field | Value |
    |-------|-------|
    | **Name** | `fluxprompt` |
    | **Type** | `url` |
    | **URL** | `https://mcp.fluxprompt.ai/mcp` |

    For the **Authorization** header, add:
    ```
    Authorization: Bearer YOUR_SERVICE_TOKEN
    ```

    Replace `YOUR_SERVICE_TOKEN` with your actual service token (starts with `mcp_`).
  </Step>
  <Step title="Save and verify">
    Save the settings. Cursor connects immediately — no restart needed.

    Open the AI chat (`Cmd/Ctrl + L`) and type:

    > *List my FluxPrompt agents*

    You should see your workspace agents listed.
  </Step>
</Steps>

## Manual config (JSON)

If you prefer editing the config file directly, Cursor's MCP config is at:

```
~/.cursor/mcp.json            (macOS / Linux)
%USERPROFILE%\.cursor\mcp.json  (Windows)
```

Add the FluxPrompt entry:

```json
{
  "mcpServers": {
    "fluxprompt": {
      "url": "https://mcp.fluxprompt.ai/mcp",
      "headers": {
        "Authorization": "Bearer mcp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
      }
    }
  }
}
```
