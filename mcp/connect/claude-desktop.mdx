## What this enables

Once connected, you can ask Claude things like:

- *"What agents do I have in FluxPrompt?"*
- *"Show me the structure of my Lead Scorer agent"*
- *"What node types are available in FluxPrompt?"*

Claude will call the FluxPrompt MCP tools automatically and show you the results inline.

<Warning>
**Claude Desktop does not accept pasted tokens.** It rejects user-supplied bearer tokens, so FluxPrompt connects via **OAuth** instead — you log in through a FluxPrompt consent screen and never handle a token. Do not try to add an `Authorization` header in a config file; it will not work. See [Authentication](/mcp/authentication).
</Warning>

## Prerequisites

- Claude Desktop installed ([download here](https://claude.ai/download))
- A FluxPrompt account with access to at least one workspace

## Step-by-step

<Steps>
  <Step title="Open the connector settings">
    In Claude Desktop, go to **Settings → Connectors** and click **Add custom connector**.
  </Step>
  <Step title="Enter the FluxPrompt server URL">
    Paste the FluxPrompt MCP server URL:

    ```
    https://mcp.fluxprompt.ai/mcp
    ```

    Give the connector a recognizable name (e.g. `FluxPrompt`) and confirm.
  </Step>
  <Step title="Log in and grant access">
    Claude opens a FluxPrompt login + consent screen. Log in with your FluxPrompt account, **select which workspace** you want to grant access to, and approve.

    No token is involved — authentication is handled entirely through this OAuth flow.
  </Step>
  <Step title="Verify the connection">
    In a new conversation, type:

    > *List my FluxPrompt agents*

    Claude should respond with a list of agents from your workspace. If you see an error, check the [Troubleshooting](/mcp/troubleshooting) page.
  </Step>
</Steps>

## What Claude can do with FluxPrompt

After connecting, Claude has access to these tools:

| Tool | What it does |
|------|-------------|
| `list_agents` | Lists all agents in your workspace |
| `get_agent` | Fetches the full structure of a specific agent |
| `list_node_types` | Lists all available FluxPrompt node types |

Claude will choose the right tool automatically based on your question. You can also ask it explicitly: *"Use `get_agent` to show me the structure of agent ID abc123"*.

## Troubleshooting

If the connection isn't working, see the [Troubleshooting](/mcp/troubleshooting) page. The most common issues are:

- Trying to use a pasted token instead of the OAuth flow (Claude Desktop only supports OAuth)
- Not completing the FluxPrompt login + consent screen
- Selecting the wrong workspace during consent
